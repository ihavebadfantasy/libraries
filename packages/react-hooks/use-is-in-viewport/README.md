# @wbe/use-is-in-viewport

## About

Check if a specific ref element is visible in viewport.

## Installation

```shell script
$ npm install @wbe/use-is-in-viewport
```

## Usage

```js
import { useIsInViewport } from "@wbe/use-is-in-viewport;";
```
